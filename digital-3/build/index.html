Laundromat Royal

  • Originality: This game is about collecting more clothes then the other team! It implements multiplayer with client-server model.
  • Technical Merit: This was way more technical than I first thought it was going to be.
    Client-Server model game using express.js and socket.io to send realtime data back and forth.
    Server hosted on AWS EC2 instance so anyone can connect over the internet.
  • Prototype Postmortem: I think I bet off a bit more than I could chew but I the tutorial I followed to set it up was easy and informative. However getting my server onto aws was a bit of a challege. Luckily I have work experience with all of these things so It made the process much smoother.
